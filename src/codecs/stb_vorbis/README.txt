stv_vorbis.c (renamed into h) version 1.22

decode ogg vorbis files from file/memory to float/16-bit signed output

-------------------
The stb_vorbis_c90.h is an equal copy of stb_vorbis.h, but all single-line
comments replaced with multiline to make the file valid for C90 standard.

Comments converted using the tool https://github.com/SwathiPrakasha/Singleline_to_multiline_comment_convertor

Please don't edit stb_vorbis_c90.h directly. Edit stb_vorbis.h first, and then,
convert it with the tool mentioned above.
